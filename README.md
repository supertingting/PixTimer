PixTimer is a simple countdown app for work or school end time, with Dynamic Island, home screen widget, and language support (English/中文).

## 常见问题 / FAQ

**Q: 如何添加主屏幕小组件？**  
**A:** 长按主屏幕 → 点击左上角「+」→ 搜索「PixTimer」→ 选择尺寸并添加。

**Q: 灵动岛不显示？**  
**A:** 请确认在「设置」中已开启「灵动岛」选项，且设备支持 Live Activity（iPhone 14 Pro 及更新机型）。

**Q: How to add the home screen widget?**  
**A:** Long press on home screen → Tap "+" → Search "PixTimer" → Choose size and add.

**Q: Dynamic Island not showing?**  
**A:** Make sure "Dynamic Island" is enabled in Settings, and your device supports Live Activity (iPhone 14 Pro and later).
